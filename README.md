### [Intersection Observer](https://www.youtube.com/watch?v=2IbRtjez6ag)

Detect dom entered viewport

```js
const observer = new IntersectionObserver((entries) => {
	entries[0].target.classList.toggle("show", entry.isIntersecting);
});
```
